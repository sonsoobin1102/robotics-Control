1주차. 자유도 자동 계산 프로그램
복잡한 기구학적 메커니즘 설계 시 필수적인 자유도(DOF) 계산을 자동화하여
설계 효율성을 높인 유틸리티 프로그램입니다.

사용 기술
 - 언어: Python
 - 수학적 모델: Kutzbach Formula

주요 기능 및 로직
 - Kutzbach Formula 기반 계산기:
   링크(Link)와 조인트(Joint)의 개수, 각 관절의 자유도 파라미터를 입력하면
   시스템 전체의 자유도를 자동으로 계산하여 출력합니다.

2~3주차. 6자유도 로봇 실시간 제어 및 특이점 감지 시뮬레이터)
6자유도(6-DOF) 로봇 팔의 정기구학(Forward Kinematics), 역기구학(Inverse Kinematics)을 동시에 사용하여 로봇을 제어하고, 자코비안(Jacobian) 행렬을 통해 위험 상태인 특이점(Singularity)을
실시간으로 감지하는 통합 시뮬레이터입니다.

사용 기술
 - 언어: Python
 - 수학적 모델: DH Parameter, Jacobian Matrix, Forward & Inverse Kinematics
 - 라이브러리: Numpy(수학적계산), PyQt5, Matplotlib, Slider(Gui & 시각화)

주요 기능 및 로직
 - 정기구학(Forward Kinematics)구현: 각 관절의 각도가 주어졌을 때 DH파라미터를 통해 로봇 끝단의 좌표를 계산하여 시각화.
 - 역기구학(Inverse Kinematics)구현: 목표 좌표를 입력하면 해당 위치로 이동하기 위한 각 관절의 회전각을 계산하여 시각화.
 - 특이점(Singularity) 판단 및 경고: 자코비안 행렬을 실시간으로 계산하여 값이 0에 가까워질 경우(로봇 제어 불능 상태) 시뮬레이션 상에 경고 표시.

4주차. GPS/IMU 센서 퓨전 기반 위치 추정 및 딥러닝 비교 분석 시뮬레이터
불확실한 GPS와 IMU 센서 데이터를 융합하여 위치 정확도를 높이는
칼만 필터(Kalman Filter) 알고리즘을 구현하고, 시계열 데이터 처리에 특화된
LSTM 딥러닝 모델과 성능을 비교 분석할 수 있는 시뮬레이터입니다.

사용 기술
 - 언어: Python
 - 수학적 모델: LSTM 딥러닝
 - 라이브러리: Matplotlib(시각화), Pytorch,pandas,scikit-learn(AI & DATA)

주요 기능 및 로직
 - Kalman Filter 센서 퓨전:
      - 예측(predict): IMU 가속도 데이터를 적분하여 물리적 이동 경로를 추정.
      - 업데이트(Update): GPS 관측 데이터를 반영하여 예측된 경로의 오차를 보정.
- LSTM 기반 경로 예측: 딥러닝 모델을 통해 전체 경로 데이터를 학습시킨구간(Train)과
  예측(Test)구간으로 분리하여, AI 모델의 일반화 성능을 시각적으로 검증.
- 사용자 상호작용: 칼만 필터의 파라미터를 조절하여 필터 성능 변화를 즉시 확인.

각 수학적 개념들
자유도: 로봇이나 기구가 공간 안에서 독립적으로 움직일 수 있는 방향의 개수.
kutzbach formula: 기구의 링크와 조인트 수로 자유도를 산출하는 공식
DG parameter: 로봇이 관절과 링크사이의 좌표변한을 수학적으로 하기 위한 4가지 표준 파라미터
Forward kinematics: 관절의 각도로 위치를 구함
Inverse Kinematics: 원하는 위치로 필요한 각도를 역산함. 
Jacobian Matrix: 관절의 회전 속도가 로봇 끝단의 이동속도에 미치는 영향을 나타내는 미분행렬
Singularity: 자코비안 행렬식이 0이 되어 로봇이 특정 방향으로 움직일 수 없는 상태.
Kalman Filter: 잡음이 포함된 측정값(GPS)과 시스템의 예측값(IMU)를 학률적 가중치(Kalman Gain)로 결합하여, 가장 참값에 가까운 상태를 추정하는 재귀적 알고리즘
Sensor Fusion: 서로 다른 특성의 센서(절대 위치 GPS vs 상대 움직임 IMU)를 결합하여 단일 센서보다 더 정확하고 신뢰성 높은 데이터를 얻는 기술입니다.
LSTM: RNN(순환신경망)의 일종으로, 긴 시간의 데이터 패턴을 기억할 수 있어 로봇의 이동경로와 같은 시계열 데이터예측에 적합한 딥러닝 모델입니다.
RNN: 이전 단계의 결과가 다음 단계의 입력으로 다시 들어가는 순환 구조의 모델
